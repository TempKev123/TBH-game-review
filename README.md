# Project Description
TBH Game Reviews is a project worked on by Sujit Dityam, Savuth Meassavrin, and Thura Kyaw. The app uploaded to Vercel (https://tbh-games.vercel.app/) is a Next.js CRUD app that can display review ratings for video games, video game publishers, and video game developers. This is a mock backend site where people with access to the site can easily change the data for a website that reviews games. (https://github.com/TempKev123/TBH-game-review.git)
